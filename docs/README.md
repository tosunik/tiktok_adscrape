## Dokümantasyon

Bu klasör, projenin **dışa açık API’lerini** (FastAPI) ve **Python tarafındaki public sınıf/fonksiyonlarını** basit bir dille anlatır.

### İçindekiler

- `API.md`: FastAPI endpoint’leri (`fastapi_server.py`)
- `PYTHON_API.md`: Public sınıf/fonksiyonlar (scraper, modeller, helper’lar)
- `CONFIG.md`: Ortam değişkenleri ve ayarlar (`src/config/settings.py`)
- `CLI.md`: Komut satırı kullanım örnekleri (`main.py`, `n8n_tiktok_scraper.py`)
- `SETUP.md`: Lokal kurulum (Python, Chrome/Chromium, çalıştırma)
